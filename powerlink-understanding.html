1 파워링크란?

핵심 메시지: 파워링크는 네이버 검색 결과 최상단에 노출되는 키워드 기반 CPC 광고입니다.
항목내용
공식 명칭사이트검색광고 (통칭: 파워링크, 키워드검색광고)
노출 위치네이버 통합검색 결과 최상단 (PC + Mobile)
과금 방식CPC (Cost Per Click) — 클릭이 발생할 때만 과금
국내 점유율검색광고 시장 65%
전국 커버리지국내 인터넷 사용자 71% 도달

파워링크의 핵심 장점

  • 구매 의도가 명확한 사용자가 직접 검색한 순간에 광고 노출
  • 노출 자체는 무료, 클릭에만 비용 발생 → 예산 효율적
  • 네이버 PC + 모바일 통합 운영, 별도 설정 없이 양쪽 동시 노출

2 캠페인 구조

계정 구조 4단계

구조도 
계정 (Account)
  └── 캠페인 (Campaign)         ← 광고 목표 단위
        └── 광고그룹 (Ad Group)  ← 전략 단위 (타겟팅/입찰/예산)
              ├── 소재 (Ad Creative)
              ├── 키워드 (Keywords)
              └── 연결 URL (Landing URL)

캠페인 (Campaign)

  • 역할: 광고의 최상위 목표 단위
  • 권장: 하나의 비즈니스 목표 = 하나의 캠페인
  • 설정 항목: 캠페인 이름, 하루 예산 한도, 기간 설정

광고그룹 (Ad Group)

  • 역할: 전략 단위 — 동일한 타겟/입찰 전략을 공유하는 키워드+소재 묶음
  • 설정 항목: 기기 타겟팅(PC/모바일), 지역 타겟팅, 요일/시간 타겟팅, 기본 입찰가

3 소재 (Ad Creative)

소재의 3가지 구성요소

구성요소설명글자 수 제한
① 제목 (Headline)광고의 핵심 메시지, 클릭을 유도하는 첫인상15자 이내
② 설명 (Description)제품/서비스 상세 설명, 행동 유도 문구45자 이내
③ 확장소재추가 정보 노출로 광고 크기와 CTR 증대유형별 상이

반응형 소재 (RSA: Responsive Search Ad)

핵심 개념: 여러 제목과 설명을 등록하면 네이버 AI가 최적 조합을 자동 선택하여 노출합니다.
구분기본 소재반응형 소재 (RSA)
제목 등록 수1개최대 15개
설명 등록 수1개최대 4개
조합 방식고정AI 자동 조합
CTR 개선기준+12% 향상

RSA 활용 팁

  • 제목 15개를 다른 메시지 각도로 작성 (가격 강조형, 혜택 강조형, 긴급성 강조형 등)
  • AI가 검색어·사용자 맥락에 맞는 최적 조합을 실시간 선택
  • 성과 데이터 누적 후 저성과 제목 교체로 지속 최적화 가능

효과적인 소재 작성 5가지 원칙

  1. 키워드 포함: 검색어를 제목에 자연스럽게 포함 → 볼드 처리되어 시선 집중
  2. 명확한 혜택 제시: "무료배송", "당일출고", "30% 할인" 등 구체적 수치 사용
  3. 차별화 포인트 강조: 경쟁사와 다른 고유한 강점 명시
  4. 행동 유도 문구(CTA): "지금 확인", "무료 상담", "바로 구매" 등 명확한 액션 유도
  5. 랜딩페이지와 일치: 광고에서 약속한 내용이 랜딩페이지에 실제로 있어야 함

확장소재 (Ad Extensions) — 13가지 유형

기본 소재(제목+설명) 외에 추가 정보를 노출해 광고 크기를 키우고 CTR을 높이는 기능입니다.

4가지 필수 확장소재 (최상단 노출 기준)

순위유형설명효과

2홍보문구브랜드/제품의 핵심 메시지 1줄 추가메시지 강화

4추가제목제목 아래 보조 제목 1줄 추가정보량 증가

확장소재 요일/시간 타겟팅

확장소재는 요일별·시간대별로 다르게 노출 설정이 가능합니다.

  • 평일 오전 → "당일 배송 가능" 홍보문구
  • 주말 → "주말 특가 이벤트" 홍보문구
  • 영업시간에만 → 전화번호 확장소재 노출

4 키워드

출처: NAVER_파워링크101_2512_FINAL.pdf 공식 분류 기준

키워드 3가지 유형

유형검색량CPC전환율(CVR)주요 목적
숏테일 (Short-tail)높음높음낮음브랜드 인지도, 최상단 노출
롱테일 (Long-tail)낮음낮음높음구매 전환, 예산 효율
시즈널 (Seasonal)기간 급등기간 상승시기 의존시즌 수요 선점

숏테일 키워드

  • 예: "운동화", "보험", "여행"
  • 검색량이 많아 노출은 크지만, 구매 의도가 불명확한 사용자 포함
  • CPC가 높아 예산 소진 빠름 → 브랜드 인지 목적으로 활용

롱테일 키워드

  • 예: "나이키 에어포스 흰색 270 남성", "30대 남성 당뇨 보험 추천"
  • 검색량은 적지만, 구매 의도가 명확한 사용자
  • CPC가 낮고 전환율이 높아 ROI 효율이 가장 우수
  • 초보 광고주에게 우선 추천

시즈널 키워드

  • 예: "발렌타인 선물 추천", "수능 이후 여행", "여름 에어컨 추천"
  • 특정 시즌·이벤트에 검색량이 급증하는 키워드
  • 시즌 2~4주 전 미리 등록하여 품질 안정화 후 노출 극대화

키워드 발굴 방법

  • 네이버 광고 시스템 내 키워드 도구: 월간 검색수, 경쟁 정도, 추천 입찰가 확인 가능
  • API 활용: /keywordstool 엔드포인트로 대량 키워드 검색량 데이터 수집 가능

5 ADVoost 검색 — AI 기반 확장 검색

출처: [네이버] ADVoost검색_플레이북_25년7월.pdf · 2026년 4월 기준: 정식 운영 중

ADVoost 검색이란?

기존 파워링크의 한계:
광고주가 직접 등록한 키워드에만 광고 노출 → 검색자가 다른 표현을 쓰면 광고 미노출
ADVoost 검색의 혁신:
네이버 AI가 검색어와 랜딩페이지 간의 연관성을 분석하여, 광고주가 등록하지 않은 키워드에도 광고를 자동으로 노출
비교 
[기존 방식]
검색어: "나이키 에어맥스 할인" → 키워드 등록 필요 → 미등록 시 광고 미노출

[ADVoost 방식]
검색어: "나이키 에어맥스 할인" → AI가 랜딩페이지 분석
→ 페이지에 관련 내용 있음 → 자동 노출

ADVoost가 SEO를 중요하게 만드는 이유

ADVoost는 크롤러가 랜딩페이지를 읽고 페이지의 주제와 검색어의 연관성을 판단합니다.

  • 랜딩페이지의 SEO가 잘 되어 있을수록 → AI가 더 많은 연관 키워드에서 광고를 노출
  • SEO가 불량한 페이지(텍스트가 이미지로만 구성, 불필요한 리다이렉트 등) → ADVoost 혜택을 받기 어려움
  • 파워링크 광고주에게 SEO는 이제 선택이 아닌 필수

6 광고 최적화 지표

2026년 4월 기준

⚠️ 품질지수(Quality Index)는 2026년에 폐지되었습니다. 아래 두 가지 신규 지표가 대체합니다.
지표의미높이는 방법
광고연관지수 검색어와 광고(소재+랜딩페이지)의 관련성 점수 키워드-소재-랜딩페이지 일관성 유지, SEO 최적화
클릭기대지수 해당 광고가 클릭될 가능성 예측 점수 소재 품질 개선, 확장소재 충실히 등록, RSA 활용
두 지수가 높을수록:
  • 동일 입찰가 대비 더 높은 순위 확보 가능
  • 실질적인 CPC 절감 효과
